# Heart Failure Prediction (Machine Learning Project)

This project explores the prediction of heart disease using machine learning techniques in Python.  
The goal is to build a model that can predict whether a patient is likely to develop heart failure based on clinical attributes such as age, blood pressure, cholesterol levels, and other health indicators.

The analysis is implemented in a Jupyter Notebook and includes:
- data exploration and preprocessing
- feature analysis
- model training and evaluation
- prediction of heart disease risk

## Dataset used:
This project uses the "Heart Failure Prediction Dataset" by fedesoriano, available on Kaggle.

Source:
https://www.kaggle.com/datasets/fedesoriano/heart-failure-prediction

Please refer to the dataset page for the original license and usage terms.
